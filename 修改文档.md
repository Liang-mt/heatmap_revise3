## 二、net.py — 网络合并 + 新增

### 原始状态
- `net.py`：大通道 UNet（64→1024），输入 80×80
- `net2.py`：小通道 UNet（8→128），输入 80×80
- `net3.py`：KeypointDetector_v2 / v2_1 / v2_heatmap，输入 128×128

### 修改内容
合并为一个 `net.py`，新增 HeatmapResUNet：

```python
from net import UNet, UNetV2, KeypointDetectorV2, KeypointDetectorV2_1, KeypointDetectorV2Heatmap, HeatmapResUNet
```

| 类名 | 来源 | 参数量 | 输入 | 输出 |
|------|------|--------|------|------|
| `UNet` | 原 net.py | 32.08M | 80×80 | 80×80 |
| `UNetV2` | 原 net2.py | 0.50M | 80×80 | 80×80 |
| `KeypointDetectorV2` | 原 net3.py | 0.56M | 128×128 | 128×128 |
| `KeypointDetectorV2_1` | 原 net3.py | 0.56M | 128×128 | 128×128 |
| `KeypointDetectorV2Heatmap` | 原 net3.py | 0.56M | 128×128 | 128×128 |
| `HeatmapResUNet` | **新增** | 1.59M | 128×128 | 128×128 |

### UNet / UNetV2 修改
接受 `num_keypoints` 参数（默认 1）：

```python
class UNet(nn.Module):
    def __init__(self, num_keypoints=1):
        self.out = nn.Conv2d(64, num_keypoints, 3, 1, 1)
```

### HeatmapResUNet 设计
参考 UNet（跳跃连接）+ ResNet（残差块）+ Simple Baseline（双线性上采样）：

```
Encoder: 3→32→64→128，每层 2 个 ResBlock
Decoder: 双线性上采样 + concat 跳跃连接 + ResBlock
Output: num_keypoints 通道，Sigmoid 激活
```

## 三、network/ — 参考网络修复

### 原始状态
8 个参考网络，大部分有问题：文件名有空格、模块级执行、import 错误、依赖 torch.hub、缺少 Sigmoid。

### 修复内容
全部重写，统一接口：

| 网络 | 参数量 | 输出尺寸 | 主要修复 |
|------|--------|----------|---------|
| `HRNet` | 1.99M | 128×128 | 上采样到 128×128，加 Sigmoid |
| `StackedHourglass` | 6.74M | 128×128 | 模块级执行移入 `__main__`，加 Sigmoid |
| `DARKpose` | 1.99M | 128×128 | 修复 import，修复 decode_heatmap 偏移 bug |
| `LEAP` | 0.13M | 128×128 | 去掉 torch.hub，自实现 InvertedResidual |
| `OpenPose` | 4.55M | 16×16 | 补全 VGG，修复通道拼接，PAF/Heatmap 分头输出 |
| `SimpleBaseline` | 15.37M | 128×128 | 去掉 torch.hub，自实现 ResNet-18 |
| `TransPose` | 4.09M | 128×128 | 去掉 torch.hub，自实现 ResNet + Transformer |
| `UNetHeatmap` | 1.86M | 128×128 | 加 Sigmoid |

统一特性：
- 全部接受 `num_keypoints` 参数
- 全部输出 Sigmoid 激活
- 全部输入 128×128
- 全部无 torch.hub 依赖（可离线运行）
- 文件名去除空格（`LEAP.py`、`SimpleBaseline.py`、`TransPose.py`、`UNetHeatmap.py`）

### OpenPose 特殊处理

## 五、train2.py — 优化训练脚本（全模型支持）

### 支持全部 14 个模型
net.py 的 6 个 + network/ 的 8 个，通过 `model_type` 参数切换。

### 基于 train.py 增加的功能

| 功能 | 说明 |
|------|------|
| 数据增强 | 翻转/旋转/亮度/噪声，训练集开启，验证集关闭 |
| PCK 精确率 | 每轮验证计算 PCK@3/5/10px + Mean Error |
| best 模型选择 | 基于 Mean Error（越低越好） |
| GT resize | GT 自动 resize 到网络原生分辨率 |
| 元组输出处理 | 自动处理 OpenPose 的 (pafs, heatmaps) 输出 |

### GT resize 逻辑
低分辨率网络（如 OpenPose 16×16）需要把 GT 下采样到网络分辨率：

```python
# 把 GT resize 到网络原生分辨率（不是把网络输出 resize 到 GT）
if outputs.shape != targets.shape:
    targets = F.interpolate(targets, size=outputs.shape[2:], mode='bilinear', align_corners=False)
```

### 训练参数
Adam、BCELoss、batch_size=16、lr=0.001、epoch=500

### PCK 精确率计算
在各 heatmap 原生分辨率取 argmax，再统一缩放到 GT 尺寸比较：

```
PCK@N = 距离 ≤ N 像素的样本比例 × 100%
Mean Error = 所有样本的平均距离（像素）
```

### 验证输出示例
```
📝 第 50 轮验证 | test_loss = 0.012345
📊 精确率     | PCK@3= 62.5% | PCK@5= 83.3% | PCK@10= 95.8% | MeanErr=3.21px
```

## 七、data.py — 数据增强 + sigma 参数化

### 原始状态
`MyDataset(path)` 无增强，sigma 硬编码为 5。

### 修改内容

1. **数据增强**：新增 `augment` 参数和 4 种增强方法
2. **sigma 参数化**：默认从 5 改为 20

```python
class MyDataset(Dataset):
    def __init__(self, path, augment=False, sigma=20):
```

### sigma 修改原因

| sigma | 128×128 峰值 | 下采样到 16×16 峰值 |
|-------|------------|-------------------|
| 5 | 1.000 | 0.550 ← 峰值模糊 |
| 10 | 1.000 | 0.845 |
| 15 | 1.000 | 0.926 |
| **20** | 1.000 | **0.958** ← 峰值清晰 |

sigma=5 下采样到低分辨率后峰值消失，模型学不到位置。sigma=20 保持峰值清晰。

### 增强方式

| 增强方式 | 概率 | 说明 |
|---------|------|------|
| 水平/垂直翻转 | 50% | 图像和坐标同步变换 |
| 旋转 ±15° | 50% | 图像和坐标同步变换 |
| 亮度/对比度 | 50% | alpha 0.8~1.2, beta ±20 |
| 高斯噪声 | 30% | std=10 |

### 单独调用增强
```python
ds.apply_single_augment(image, points, 'flip')
ds.apply_single_augment(image, points, 'rotate')
ds.apply_single_augment(image, points, 'brightness')
ds.apply_single_augment(image, points, 'noise')
```

## 八、predict.py / test.py — 推理脚本

### 修改内容
1. **导入**：统一从 net.py + network/ 导入全部模型
2. **权重加载**：兼容新旧两种格式
3. **元组输出处理**：自动处理 OpenPose
4. **坐标提取**：原始分辨率取 argmax + 缩放（避免插值误差）

```python
# OpenPose 返回元组
if isinstance(out, tuple):
    out = out[1][:, -1]

# 原始分辨率取 argmax，再缩放到原图尺寸
hm_h, hm_w = out.shape[-2], out.shape[-1]
idx = out.argmax().item()
hy, hx = divmod(idx, hm_w)
x = int(hx * img.width / hm_w)
y = int(hy * img.height / hm_h)
```

### predict.py 顶部配置
```python
model_type = 'KeypointDetectorV2'

```
